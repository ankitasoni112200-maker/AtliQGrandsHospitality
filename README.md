# 🏨 Hotel Insights Dashboard (AtliQ Grands)

## 📊 Project Overview

- This project presents a Power BI dashboard built to analyze hotel performance data for AtliQ Grands.

- The goal is to uncover key insights related to revenue, occupancy, pricing strategies, and customer behavior to support data-driven decision-making.

# 🎯 Key Objectives

- Analyze Revenue, RevPAR, ADR, and Occupancy
- Identify top-performing cities & properties
- Evaluate the booking platforms' performance
- Understand customer ratings' impact on revenue
- Provide business recommendations

## 📌 Key Metrics (KPIs)

- 💰 Revenue: ₹1.69 Billion

- 🏨 RevPAR: 7337

- 📈 Occupancy Rate: 57.79%

- 💵 ADR: ₹12.70K

- 📊 Realisation %: 70.14%

- 🛏️ Total Bookings: 133K

## 📈 Insights & Findings

- Mumbai generates the highest revenue (39.1%)

- Delhi contributes the lowest revenue (17.2%)

- Weekends have higher occupancy than weekdays

- Luxury rooms contribute most revenue

- Direct offline bookings are lowest (5%)

- Strong correlation between ratings and revenue

- Other booking platforms contribute ~40% bookings

## 🛠️ Tools & Technologies
- Power BI
- DAX (Data Analysis Expressions)
- Data Modeling
- Excel / CSV Dataset

## 📂 Dataset

- Includes:
- Fact Tables (Bookings, Revenue)
- Dimension Tables (Hotels, Rooms, Cities)

## 🚀 How to Use
1. Download the dataset (CSV files)
2. Import data into MySQL for cleaning
3. Load cleaned data into Power BI
4. Create relationships between tables
5. Build DAX measures
6. Design dashboard visuals
7. Explore insights via visuals
